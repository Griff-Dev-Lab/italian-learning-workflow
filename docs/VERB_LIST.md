## 📚 Learning Strategy

### Phase 1: Master Tier 1 (Week 1–2)
- Generate flashcards for all 10 core verbs
- Import into Anki and review daily
- Focus on present, past, and future tenses
- Goal: Automatic recall of all forms

### Phase 2: Add Tier 2 (Week 3–4)
- Generate flashcards for all 10 everyday verbs
- Add to your Anki deck
- Review Tier 1 + Tier 2 together
- Goal: Fluent use in daily conversations

### Phase 3: Expand with Tier 3 (Week 5–6)
- Generate flashcards for all 10 descriptive verbs
- Complete your 30-verb foundation
- Review all three tiers regularly
- Goal: Confident communication about your life

2. **Use the `--table` flag** — Print conjugation tables for quick reference
3. **Review daily** — Spaced repetition is key to retention
4. **Practice speaking** — Use the verbs in sentences, not just flashcards
5. **Add context** — Create your own example sentences using these verbs
