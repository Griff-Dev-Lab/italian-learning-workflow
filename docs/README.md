## 🚀 Getting Started

1. **Read** [VERB_LIST.md](VERB_LIST.md) to understand the learning tiers
2. **Generate** flashcards for Tier 1 verbs using the provided CLI commands
3. **Follow** [ANKI_SETUP.md](ANKI_SETUP.md) to import into Anki
4. **Study** daily using spaced repetition
5. **Progress** through Tier 2 and Tier 3 as you master each level
